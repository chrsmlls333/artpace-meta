# Artpace Metadata Creator

### Installation
- Install Homebrew https://brew.sh/
- `brew install node`
- `brew install mediainfo`
- `brew install richardlehane/digipres/siegfried `
- `npm install`
- `npm link` 
  - registers the command `artpace-meta` to the system

### Commands
- `artpace-meta --help`
- `artpace-meta define [source]`
